RunString Method

Runs phreeqc using the given string as input. Returns the number of errors encountered during the run.

expression.RunString( Input )

expression    Required. An expression that returns an IPhreeqcCOM object.

Input   Required String. The string containing phreeqc input.

Remarks

The RunString method cannot be called until a database has been successfully loaded by one of the LoadDatabase methods: LoadDatabase or LoadDatabaseString.

Example

This example stores phreeqc input into a string and runs it with the IPhreeqcCOM object.


  JScript

var iphreeqc = new ActiveXObject("IPhreeqcCOM.Object");
iphreeqc.LoadDatabase("phreeqc.dat");

var sInput = "SOLUTION 1\n";
sInput += "temp 25.0\n";
sInput += "pH   7.0\n";

iphreeqc.OutputFileOn = true;
iphreeqc.RunString(sInput);
WScript.Echo("see phreeqc.out");

  VBScript

Dim iphreeqc, sInput
Set iphreeqc = CreateObject("IPhreeqcCOM.Object")
iphreeqc.LoadDatabase("phreeqc.dat")

sInput = "SOLUTION 1" & vbNewLine
sInput = sInput & "temp 25.0" & vbNewLine
sInput = sInput & "pH   7.0" & vbNewLine

iphreeqc.OutputFileOn = True
iphreeqc.RunString(sInput)
WScript.Echo("see phreeqc.out")
