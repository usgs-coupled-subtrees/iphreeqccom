LoadDatabase Method

Loads a phreeqc database into the IPhreeqcCOM object. Returns the number of errors encountered while loading the database.

expression.LoadDatabase(FileName )

expression    Required. An expression that returns an IPhreeqcCOM object.

FileName   Required. The file name of the phreeqc database to load.

Example

This example loads the "phreeqc.dat" database into the IPhreeqcCOM object.

Set iphreeqc = CreateObject("IPhreeqcCOM.Object")
iphreeqc.LoadDatabase "C:\Program Files\USGS\phreeqc-2.12.1\phreeqc.dat"
