SelectedOutputFileOn Property

True to turn on the selected-output file. The file will be named selected.out  unless a FileName was specified within the SELECTED_OUTPUT keyword data block. The initial setting is False. Read/write Boolean.

Remarks

The SelectedOutputFileOn property is used only by the Run methods: RunAccumulated, RunFile and RunString.

Example

This example runs ex2  with the selected-output file turned on.


  JScript

var iphreeqc = new ActiveXObject("IPhreeqcCOM.Object");
iphreeqc.LoadDatabase("phreeqc.dat");
iphreeqc.SelectedOutputFileOn = true;
iphreeqc.RunFile("ex2");
WScript.Echo("see ex2.sel");


  VBScript

Dim iphreeqc
Set iphreeqc = CreateObject("IPhreeqcCOM.Object")
iphreeqc.LoadDatabase("phreeqc.dat")
iphreeqc.SelectedOutputFileOn = True
iphreeqc.RunFile("ex2")
WScript.Echo("see ex2.sel")
