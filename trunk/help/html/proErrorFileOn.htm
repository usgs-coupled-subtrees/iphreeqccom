ErrorFileOn Property

True to turn on output to the phreeqc.err  file . The initial setting is False. Read/write Boolean.

Remarks

The ErrorFileOn property is used only by the Run methods: RunAccumulated, RunFile and RunString.

Example

This example runs ex2  with the phreeqc.err  file turned on.

The example attempts to run the ex1  input file, fails, and writes the error messages to the phreeqc.err  file(no database is loaded).


  JScript

try {
    var iphreeqc = new ActiveXObject("IPhreeqcCOM.Object");
    iphreeqc.ErrorFileOn = true;
    iphreeqc.RunFile("ex1");
}
catch(e) {
    WScript.Echo("See phreeqc.err");
}

  VBScript

Dim iphreeqc
On Error Resume Next
Set iphreeqc = CreateObject("IPhreeqcCOM.Object")
iphreeqc.ErrorFileOn = True
If (iphreeqc.RunFile("ex1") <> 0) Then
    WScript.Echo("See phreeqc.err")
End If
