LoadDatabaseString Method

Loads a phreeqc database into the IPhreeqcCOM object. Returns the number of errors encountered while loading the database.

expression.LoadDatabaseString( Input )

expression    Required. An expression that returns an IPhreeqcCOM object.

Input   Required String. The string containing phreeqc database.

Example

This example reads the phreeqc.dat  database into a String and loads it into the IPhreeqcCOM object.


  JScript

var ForReading = 1

var fso = new ActiveXObject("Scripting.FileSystemObject");
var ts = fso.OpenTextFile("phreeqc.dat", ForReading);
var strDB = ts.ReadAll();

var iphreeqc = new ActiveXObject("IPhreeqcCOM.Object");
if (iphreeqc.LoadDatabaseString(strDB) == 0) {
    WScript.Echo("database ok");
} else {
    WScript.Echo("database contains errors");
}


  VBScript

Const ForReading = 1

Dim fso, ts, strDB, Phreeqc

Set fso = CreateObject("Scripting.FileSystemObject")
Set ts = fso.OpenTextFile("phreeqc.dat", ForReading)
strDB = ts.ReadAll()

Set iphreeqc = CreateObject("IPhreeqcCOM.Object")
If (iphreeqc.LoadDatabaseString(strDB) = 0) Then
    WScript.Echo("database ok")
Else
    WScript.Echo("database contains errors")
End If
