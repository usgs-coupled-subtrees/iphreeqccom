Lines Property

Returns the accumulated text in the input buffer of the IPhreeqcCOM object. Read-only String.

Example

This example loads some keyword input into the IPhreeqcCOM object and displays the result.


  JScript

var iphreeqc = new ActiveXObject("IPhreeqcCOM.Object");
iphreeqc.AccumulateLine("SOLUTION 1");
iphreeqc.AccumulateLine("END");
WScript.Echo("The accumulated input is:\n" + iphreeqc.Lines);

  VBScript

Dim iphreeqc
Set iphreeqc = CreateObject("IPhreeqcCOM.Object")
iphreeqc.AccumulateLine("SOLUTION 1")
iphreeqc.AccumulateLine("END")
WScript.Echo("The accumulated input is:" & vbNewLine & iphreeqc.Lines)
