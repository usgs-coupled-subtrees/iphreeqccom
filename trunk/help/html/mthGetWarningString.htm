GetWarningString Method

Returns a string containing any warning messages that were generated during the last invocation of the following methods: LoadDatabase, LoadDatabaseString, RunAccumulated, RunFile, RunString.

expression.GetWarningString()

expression    Required. An expression that returns an IPhreeqcCOM object.

Example

The example loads the phreeqc.dat  database and attempts to use the DATABASE keyword to set the database to wateq4f.dat. A warning is displayed stating that the DATABASE keyword is ignored by IPhreeqcCOM.


  JScript

var iphreeqc = new ActiveXObject("IPhreeqcCOM.Object");
iphreeqc.LoadDatabase("phreeqc.dat");
iphreeqc.AccumulateLine("DATABASE wateq4f.dat");
iphreeqc.AccumulateLine("SOLUTION 1");
iphreeqc.RunAccumulated();
if (iphreeqc.GetWarningString().length != 0) {
    WScript.Echo(iphreeqc.GetWarningString());
}


  VBScript

Dim iphreeqc
Set iphreeqc = CreateObject("IPhreeqcCOM.Object")
iphreeqc.LoadDatabase("phreeqc.dat")
iphreeqc.AccumulateLine("DATABASE wateq4f.dat")
iphreeqc.AccumulateLine("SOLUTION 1")
iphreeqc.RunAccumulated()
If (Len(iphreeqc.GetWarningString()) <> 0) Then
    WScript.Echo(iphreeqc.GetWarningString())
End If
