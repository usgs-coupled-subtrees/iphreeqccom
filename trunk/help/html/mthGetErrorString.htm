GetErrorString Method

Returns a string containing any error messages that were generated during the last invocation of the following methods: AccumulateLine, GetSelectedOutputValue, LoadDatabase, LoadDatabaseString, RunAccumulated, RunFile, RunString.

expression.GetErrorString()

expression    Required. An expression that returns an IPhreeqcCOM object.

Example

The example attempts to run the ex1  input file, fails, and displays the reason for failure (no database is loaded).


  JScript

try {
    var iphreeqc = new ActiveXObject("IPhreeqcCOM.Object");
    iphreeqc.RunFile("ex1");
}
catch(e) {
    WScript.Echo(iphreeqc.GetErrorString());  // could also use e.message
}

  VBScript

Dim iphreeqc
On Error Resume Next
Set iphreeqc = CreateObject("IPhreeqcCOM.Object")
If (iphreeqc.RunFile("ex1") <> 0) Then
    WScript.Echo(iphreeqc.GetErrorString())
End If
