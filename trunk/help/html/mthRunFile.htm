RunFile Method

Runs phreeqc using the given file as input. Returns the number of errors encountered during the run.

expression.RunFile( FileName )

expression    Required. An expression that returns an IPhreeqcCOM object.

FileName   Required String. The name of the phreeqc input file to run.

Remarks

The RunFile method cannot be called until a database has been successfully loaded by one of the following methods: LoadDatabase or LoadDatabaseString.

Example

This example runs the ex1  input file in the IPhreeqcCOM object.


  JScript

var iphreeqc = new ActiveXObject("IPhreeqcCOM.Object");
iphreeqc.LoadDatabase("phreeqc.dat");
iphreeqc.OutputFileOn = true;
if (iphreeqc.RunFile("ex1") == 0) {
    WScript.Echo("see phreeqc.out");
}

  VBScript

Set iphreeqc = CreateObject("IPhreeqcCOM.Object")
iphreeqc.LoadDatabase("phreeqc.dat")
iphreeqc.OutputFileOn = True
If (iphreeqc.RunFile("ex1") = 0) Then
    WScript.Echo("see phreeqc.out")
End If
